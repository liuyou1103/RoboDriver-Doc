# 核心介绍

`RoboDriver`项目是智源基于[LeRobot](https://github.com/huggingface/lerobot.git)的全面改进版本，改进并扩展了数据集格式、机器人接入方式，同时提供了智源开发的`RoboX-Studio 具身一站式开发平台`的接入能力。我们将在未来兼容国内标准，更加方便行业内机器人接入。

## 价值 改进 扩展
1. **数据集体系升级**：全面改进并扩展 LeRobot 原生数据集格式，适配多模态数据采集（如深度相机、RGB 相机、触觉传感器等），支持数据独立存储、编辑与标准化导出，兼容改进版 LeRobot 数据集规范；
2. **机器人接入能力增强**：优化多类型机器人的接入方式，降低开源/商用机械臂、移动机器人等设备的接入门槛，提供开箱即用的部署指南与二次开发示例；
3. **平台生态整合**：无缝对接智源自研的 `RoboX-Studio 具身智能一站式开发平台`，实现机器人驱动、数据采集、模型训练、任务编排的全流程打通；
4. **本土化标准兼容**：规划兼容国内机器人行业通用标准，未来将进一步降低国内厂商/开发者的接入成本，推动行业标准化落地。

RoboDriver 秉持「One Driver. Every Robot.」的理念，通过统一的驱动层抽象与标准化接口，让开发者无需关注不同机器人的底层实现，聚焦于业务逻辑与算法开发，大幅提升机器人应用研发效率。
