RoboDriver's Dora ecosystem has a new member! The **SO101 open-source lightweight arm** adapter repo is live, supporting 6-DOF plus gripper control, with dual RealSense depth + RGB capture, fully compatible with RoboDriver's device driver and data pipelines.
<!-- truncate -->

The SO101 Dora adapter offers an out-of-the-box guide so you can connect and start collecting quickly:
Link: [robodriver-robot-so101-aio-dora](https://github.com/BAAI-EI-DATA/robodriver-robot-so101-aio-dora)

Highlights:
- Low-latency command and data transport built on Dora
- Per-sample storage and editing, aligned with the extended LeRobot dataset format
- Sample extensions and docs for open-source arm secondary development
